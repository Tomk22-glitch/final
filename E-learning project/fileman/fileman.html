Part 1: Create Folders

Main Folder

  • On the desktop, create a new folder named My Files.

Subfolders

  • Inside My Files, create three folders:
    • School
    • Work
    • Personal

Part 2: Save Documents in Different Formats

Text File

  • Open Notepad (Windows) or TextEdit (Mac).
  • Type: "My name is [Your Name]. I am learning computer skills."
  • Save in Personal folder as:
    • AboutMe.txt
    • AboutMe.rtf

Word Document

  • Open Microsoft Word or Google Docs.
  • Write: "My favorite subject is [Subject]."
  • Save in School folder as:
    • Subject.docx
    • Subject.pdf

Part 3: Manage Files

  • Move & Rename: Move AboutMe.txt from Personal to School. Rename it to Intro.txt.
  • Delete & Recover: Delete AboutMe.rtf. Recover from Recycle Bin/Trash if needed.

Part 4: Extra Practice

  • Create Shortcut: Right-click My Files → Send to Desktop.
  • Search: Open My Files and search for .pdf files.

Assessment Checklist

  • Correct folder structure (My Files → School, Work, Personal)
  • Files saved in proper formats (.txt, .rtf, .docx, .pdf)
  • Files moved/renamed correctly
  • Understanding of basic file operations
