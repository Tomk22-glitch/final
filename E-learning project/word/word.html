1. Definition of Word Processing

A word processor is application software used to create, edit, format, and print documents. Examples include Microsoft Word and Google Docs, supporting real-time collaboration and advanced editing.

3. Keyboard Shortcuts

Standard shortcuts for Windows (Command for macOS):

  • Bold: Ctrl + B
  • Italic: Ctrl + I
  • Underline: Ctrl + U
  • Save: Ctrl + S
  • Undo: Ctrl + Z
  • Redo: Ctrl + Y
  • Select All: Ctrl + A
  • Copy: Ctrl + C
  • Cut: Ctrl + X
  • Paste: Ctrl + V

4. Printer Types
