Computer Studies Notes

1. Define the following terms:

  • Computer: An electronic device that processes raw data using predefined instructions to produce meaningful information.
  • Data: Raw facts or figures needing processing.
  • Information: Organized and meaningful data.
  • ICT: Integration of digital tech for communication and data handling.

2. Basic Components of a Computer System

  • Hardware: Physical parts like CPU, monitor.
  • Software: Operating systems and applications.
  • Users: People using the system.
  • Data: Inputs and outputs.
  • Networking: Routers, modems for connectivity.

3. First General-Purpose Computer

  • ENIAC: Built in 1945 for military calculations, paved the way for modern computing.

4. Acronyms and Their Functions

  • RAM: Temporary working memory.
  • ROM: Permanent memory for firmware.
  • UPS: Backup power during outages.
  • CPU: Main processing unit.
  • USB: Standard for peripherals.
  • HDD: Magnetic permanent storage.
  • SSD: Fast storage with no moving parts.

5. Generations of Computers

  • 1st: Vacuum tubes.
  • 2nd: Transistors.
  • 3rd: Integrated Circuits.
  • 4th: Microprocessors.
  • 5th: AI, Quantum computing.

6. Parts of a Computer System

Hardware

  • Input: Keyboard, mouse.
  • Processing: CPU, GPU.
  • Memory: RAM, ROM.
  • Storage: HDD, SSD.
  • Output: Monitor, printer.
  • Motherboard: Connects all parts.
  • PSU: Power Supply Unit.

Software

  • System Software: OS like Windows.
  • Application Software: MS Office, browsers.

7. Major Applications of Computers

  • Education: E-learning, online exams.
  • Healthcare: Records, telemedicine.
  • Business: E-commerce, payroll.
  • Transport: GPS, traffic control.
  • Entertainment: Streaming, gaming.

8. Classification of Computers

By Size

  • Supercomputers
  • Mainframes
  • Microcomputers

By Purpose

  • General-purpose
  • Special-purpose

By Functionality

  • Analog
  • Digital
  • Hybrid

9. Advantages of Computers

  • Speed
  • Accuracy
  • Storage
  • Automation
  • Connectivity

10. Computer Laboratory

  • A dedicated space for hands-on ICT training.

11. Computer Lab Safety Precautions

  • No food/drinks near devices.
  • Use surge protectors.
  • Good cable management.
  • Update antivirus regularly.
  • Use ergonomic setups.
  • Keep fire extinguishers handy.

12. Types of UPS Systems

  • Standby UPS
  • Line-Interactive UPS
  • Online UPS

13. Cybersecurity

  • Definition: Protect systems from threats.
  • Threats: Malware, phishing, hacking.

System Software Application Software

15. Computer Operations

  • Warm Booting: Restart.
  • Shutting Down: Safe close.
